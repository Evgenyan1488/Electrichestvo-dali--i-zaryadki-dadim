Эта программа демонстрирует работу ранжирования списка районов и домов по необходимости установки зарядной станции. 

Программа вычисляет в какой район и в какой дом нужно установить зарядную станцию в первую очередь, чтобы польза была максимальна. 

Она учитывает несколько критериев. 

При ранжировании домов: количество жителей с электромобилями в доме и расстояние от дома до пс. 
При ранжировании районов: резерв мощности в каждом районе, количество уже установленных зарядок и общее количество жителей с электромобилями. 

В начале в каждом районе программа вычисляет в какой дом в первую очередь необходимо поставить зарядку основываясь на вычисляемом коэффициенте. Программа получает входные данные, переводит их в относительные единицы и вычисляет коэффициент. Далее сортирует по убыванию коэффициента. Чем выше коэффициент, тем больше подходит дом для установки зарядки. 

Далее внутри района программа вычисляет в какой район вначале выгоднее ставить зарядные станции.  
Программа получает входные данные, переводит их в относительные единицы и вычисляет коэффициент. Далее сортирует по убыванию коэффициента. Чем выше коэффициент, тем больше подходит район для установки зарядки. 

На выходе программа выдает список районов по убыванию выгоды установки зарядной станции в каждом районе. 

На данный момент ввод и вывод данных происходит с консоли, но в дальнейшем планируется подключение базы данных для работы с данными.
